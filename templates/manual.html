Panduan Penggunaan Website Scraping

Langkah 1: Login (Penting!)

Beberapa dokumen (PDF) di repository bersifat restricted dan hanya bisa diakses setelah login. Gunakan fitur Login di halaman utama untuk masuk menggunakan akun repository Anda. Sistem akan membuka browser secara otomatis, melakukan login, dan menyimpan sesi Anda agar proses scraping bisa mengakses semua file.

Langkah 2: Pilih Mode Pencarian

Ada dua cara utama untuk melakukan scraping:

A. ðŸ”Ž Halaman Daftar Judul (List)

Gunakan mode ini untuk mencari kata kunci di banyak dokumen sekaligus dari halaman daftar subjek.

  • Pilih Jurusan: Pilih fakultas atau jurusan yang ingin Anda cari dari daftar dropdown.
  • Pilih Anchor (Tahun): (Opsional) Pilih tahun spesifik untuk memulai pencarian dari tahun tersebut ke bawah. Berguna jika Anda hanya ingin mencari skripsi dari tahun-tahun terbaru.
  • Masukkan Kata Kunci: Tulis kata atau frasa yang ingin Anda cari di dalam file PDF.
  • Indeks Mulai: Angka urutan untuk memulai pencarian. Biarkan 0 untuk memulai dari paling awal.
  • Batas Halaman: Jumlah maksimum dokumen yang akan diperiksa. Semakin besar angkanya, semakin lama prosesnya.

B. ðŸ“„ Langsung ke Halaman Judul (Detail)

Gunakan mode ini jika Anda sudah tahu halaman dokumen spesifik yang ingin diperiksa.

  • URL Halaman Detail: Masukkan URL lengkap halaman dokumen, contohnya: https://repository.upnjatim.ac.id/16186/.
  • Masukkan Kata Kunci: Tulis kata kunci yang ingin Anda cari di dalam PDF pada halaman tersebut.

Langkah 3: Membaca Hasil

Setelah Anda memulai proses scraping, Anda akan diarahkan ke halaman hasil yang menampilkan progres secara real-time.

  • Panel Hasil Ditemukan: Menampilkan daftar halaman dokumen di mana kata kunci Anda ditemukan. Di bawah setiap halaman, akan ada link menuju file PDF beserta jumlah kemunculan kata kunci di dalamnya.
  • Panel Log Proses: Menampilkan detail langkah-langkah yang sedang dilakukan oleh sistem, seperti "Memeriksa URL...", "Mendownload PDF...", dll. Ini membantu Anda memantau proses yang berjalan.

Proses akan ditandai sebagai selesai jika semua item dalam batas yang Anda tentukan sudah diperiksa.
